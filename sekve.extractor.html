sekve.extractor.ss_extractor module

sekve.extractor.ss_extractor.extract_ss(v: ndarray, i: ndarray, t: float, exp_val: Optional[Union[list, str]] = 'auto', method: str = 'savgol', frac: float = 0.2, max_id: Optional[float] = None, min_id: Optional[float] = None, take_val: str = 'min', show: bool = False, return_crit_v: bool = False, ylim=(0, 200)) → Union[float, tuple]

Extract subthreshold swing.

Parameters

  • v – gate voltage..

  • i – drain current.

  • t – temperature in Kelvin.

  • exp_val – Optional. Expected value of ss depending on temperature. [[t1, t2, ss1, ss2], [], …]

  • method – Optional, [‘savgol’, ‘poly’]

  • frac – Optional. Fraction of filter windows data length to entire data length. This parameter only attributes to method ‘savgol’.

  • max_id – Optional. Upper limitation of current.

  • min_id – Optional. Lower limitation of current.

  • take_val – Optional. [“min”, “avg”]

  • show – If Ture, show the extraction.

  • return_crit_v – If True, return critical voltage.

  • ylim – limitation of ss in mV/dec
